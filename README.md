# 🎓 Student Grade Manager

A Python command-line application that tracks subject marks for a student, evaluates pass/fail statuses, calculates highest/lowest scores, and formats output using modern Python string formatting.

---

## ✨ Features

* **Dynamic Initials Generator:** Uses a `for` loop to dynamically extract uppercase initials regardless of name length.
* **Formatted Subject List:** Iterates through a tuple of subjects with aligned terminal formatting using f-strings and escape sequences.
* **Interactive Marks Entry:** Collects marks for each subject and stores them in a synchronized dictionary.
* **Aligned Marks Display:** Uses advanced f-string alignment specifiers (e.g., `{sentence:<27}`) to display subjects and scores in a clean column format.
* **Pass/Fail Evaluation:** Evaluates individual subject thresholds and uses `for-else` loop logic to issue global pass/warning messages.
* **Performance Analytics:** Calculates highest score, lowest score, and counts unique marks via Python `sets`.
* **Birth Year Calculation:** Computes the user's estimated birth year from their age input.

---

## 🛠️ Python Concepts Demonstrated

* **Loops & Control Flow:** `for` loops, `range()`, `break`, and the `for-else` structure.
* **Data Structures:** Dictionaries, Tuples, Lists, and Sets (`set()`).
* **String Formatting:** Modern f-strings, alignment specifiers (`<27`), `.title()`, `.upper()`, and `.split()`.
* **Built-in Functions:** `sum()`, `len()`, `max()`, and `min()`.
